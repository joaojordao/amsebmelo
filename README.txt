# AMSM — Static HTML/CSS
Como usar:
1) Envie os ficheiros para o seu alojamento (ex.: cPanel, Netlify, GitHub Pages, Vercel).
2) A página inicial é `index.html`. O estilo está em `styles.css`.
3) Para alterar cores/tipografia, edite as variáveis no topo do CSS.

SEO rápido:
- Edite a `<title>` e a `<meta name="description">` conforme necessário.
